Sea Battle is currently in **Coming Soon** state.

## Planned mode structure

- VS AI
- Online 1v1

## Draft gameplay loop

1. Place units privately.
2. Attack by coordinate each turn.
3. Read hit/miss feedback.
4. Destroy all key enemy units to win.

## Current focus before release

- Better combat feedback and board readability
- Mobile-first targeting controls
- Stable reconnect and sync behavior for online rooms
